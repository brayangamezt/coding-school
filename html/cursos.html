Hola que tal, bienvenido a coding schoool, el lugar perfecto

donde encontraras una gran variedad de cursos para convertirte

en un desarrollador web backend o frontend

Curso python: Aprende python y conviertete en desarrollador web

Python es actualmente el lenguaje de programacion mas popular y en este curso aprenderas todo lo necesario para convertirte en un desarrollador web. Aprenderas a programar tanto en frontend como en el backend de una pagina web utilizando un framework como django. Tambien aprenderas a desarrollar aplicaciones de escritorio utilizando tkinter

Curso HTML:El curso mas completo de HTML desde cero

Unete al curso mas completo de HTML solo haciendo click aqui. Encontraras todo lo que hay que saber para dominar HTML5, creando tus primeras paginas web y poder iniciarte en el mundo de el desarrollo web. Aprenderas todo lo que hay que saber sobre etiquetas y estructuras de una pagina web, ya que todo sitio o aplicacion web lo utiliza.

Curso css:Aprende css de la A a la Z

En este curso podras aprender todo lo relacionado con CSS, se abarcan todo lo relacionado con el diseño de una pagina web, como lo son, colores, formas, animaciones, responvsive desgin, sombras, fuentes etc. No pierdas la oportunidad de formar parte de este gran curso y poder iniciarte en el mundo del desarrollo y diseño web

Curso PHP:Master en PHP8 y sus novedades

Toma el curso mas completo en PHP8, el lenguaje mas popular y utilizado para el desarrollo backend. PHP es uno de los lenguajes mas usados en la web, por muchos motivos, uno de ellos su ecosistema de tecnologias a su alrededor. Frameworks, librerias, CMS´S, una comunidad muy amplia, demanda laboral elevada han convertido a PHP en unos de los mejores lenguajes de backend.

Curso JS:El lenguaje mas porpular en desarrollo web

Aprende todo lo que necesitas sobre el lenguaje de programacion mas utilizado para el desarrollo web. En este curso veremos javascript desde cero, conociendo como utilizar una variable, tipos de datos, condiciones, creacion de etiquetas HMTL desde JS, POO etc. Tambien aprenderemos a utilizar librerias como jquery y gsap para crear increibles animaciones.

SISTEMAS OPERATIVOS

Windows

Aprender como utilizar utilizar windows para convertirte en un desarrollador. En este curso te enseñaremos como podras instalar diferentes tecnologias dentro de tu sistema operativo windows utilizando tu consola de comandos.

Linux

GNU/linux actualmente esta presente en miles de millones de dispostivos, si no tienes nada de conocimientos base de GNU/Linux, no te preocupes, este curso es para ti ya que aprenderemos desde cero como usar de forma grafica varios enternos de escritorio y linea de comandos.

MacOs

Aprende sobre el nuevo sistema de apple de cero a experto de forma facil y sencilla con Mac Os y las herramientas mas profesionales del momento. A lo largo del curso tendras varias hojas de ejemplos practicos para que captes bien el sentido de cada concepto.
